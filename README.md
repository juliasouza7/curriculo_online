# Currículo Virtual — Júlia Costa

Site estático criado para apresentar meu currículo, experiências, projetos, habilidades, certificados e formas de contato em um formato mais visual e acessível.

## Sobre o projeto

Este projeto funciona como uma versão online do meu currículo, reunindo informações profissionais em uma página simples, responsiva e organizada por seções.

A ideia é centralizar em um único lugar:
- Resumo profissional
- Experiências
- Formação
- Habilidades técnicas
- Projetos pessoais
- Certificados
- Links de contato

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Funcionalidades

- Navegação por seções
- Exibição de certificados
- Links para projetos no GitHub
- Download/visualização do currículo em PDF
- Layout responsivo
- Favicon personalizado

## Projetos em destaque

### Loja de Material de Construção

Sistema simples para controle de produtos e vendas de uma loja de material de construção, utilizando uma planilha Excel como base de dados.

Repositório:  
https://github.com/juliasouza7/loja_material_construcao

## Como visualizar

O projeto pode ser publicado via GitHub Pages.

Também é possível executar localmente abrindo o arquivo `index.html` no navegador.
